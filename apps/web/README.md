# Slackbracket Web (Next.js)

Modernized frontend target for Slackbracket.

## Highlights
- Next.js App Router + TypeScript
- Shared domain package (`@slackbracket/domain`)
- Sparse-pick-first UX with quick generate and guided custom modes
- Live odds panel (`1 in N` + exponent framing)
- Live tournament lock integration
- TheSportsDB logo adapter with polished placeholders

## Run
- `npm install`
- `npm run dev`
